Fundamentos CSS

Definición de CSS

CSS (Cascading Style Sheets) es un lenguaje de diseño que se utiliza para definir cómo se ven las páginas web. Permite controlar el color, la fuente, la disposición de los elementos, las animaciones y los efectos visuales.

Selectores de CSS

  1. Etiquetas: menos específicas.
  2. Clases: recomendadas para CSS - Especificidad intermedia.
  3. Identificadores únicos: más específico
